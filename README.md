# gig

## Usage

```
gig clone kaneshin/gig
```

### Over SSH

```
gig clone kaneshin/gig -p ssh
```

### Clone from BitBucket

```
gig clone kaneshin/gig -h bitbucket
```

## Requirements

- Git

## Installation

```
go get -u github.com/kaneshin/gig
```
